Descripción del Sistema

Esta aplicación web fue diseñada como solución de fin de curso para la asignatura de Inteligencia Artificial. Ofrece una infraestructura genérica que permite cargar conjuntos de datos en formato CSV, imputar de forma automatizada valores nulos, codificar variables textuales/categóricas, y entrenar clasificadores de Machine Learning basados en la Regresión Logística.

Tecnologías Implementadas

Python 3 Flask (Web Server) Pandas (Data Science) Scikit-Learn (ML Engine) Matplotlib (Visualization) Custom CSS Grid Layouts

Modelos e Inferencia

  • Regresión Logística: Un clasificador estadístico lineal adecuado para modelar variables categóricas mediante la función sigmoide.
  • Codificación Dinámica: Implementa transformaciones a través de LabelEncoder que mapean texto a valores numéricos enteros, guardando y persistiendo los mapeos en archivos serializados (.pkl) para inferencias futuras.

Métricas de Evaluación Reportadas

  • Accuracy: Porcentaje total de predicciones correctas sobre el conjunto de test.
  • Precision: Capacidad del modelo para evitar falsos positivos (ponderación por clase).
  • Recall: Capacidad del modelo para identificar todas las instancias positivas (sensibilidad).
  • F1 Score: Media armónica entre Precision y Recall.
  • Matriz de Confusión: Representación bidimensional de predicciones estimadas frente a los valores reales.
