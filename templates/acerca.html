V2.0

Descripción del Sistema

Esta aplicación web fue diseñada como solución de fin de curso para la asignatura de Inteligencia Artificial. Ofrece una infraestructura genérica que permite cargar conjuntos de datos en formato CSV, imputar de forma automatizada valores nulos, codificar variables categóricas, y entrenar clasificadores basados en la Regresión Logística.

Tecnologías Implementadas

Python 3 Flask (Web Server) Pandas (Data Science) Scikit-Learn (ML Engine) Matplotlib (Visualization) Thread Safety (Locks)

Pipeline de Datos e Inferencia

  • Regresión Logística: Clasificador estadístico lineal adecuado para modelar clases mediante la función sigmoide y umbrales de decisión.
  • Imputación Inteligente: Relleno automático de valores nulos (mediana para numéricas, moda para categóricas) y discretización automática mediante pd.cut() para variables objetivo continuas.
  • Persistencia de Estado (joblib): Serialización en disco (archivos .pkl) para cargar el modelo, los codificadores y las variables dinámicamente y predecir sin perder estado al recargar la web.
  • Thread Safety (Seguridad en Concurrencia):** Implementa un Thread Lock (matplotlib_lock) para procesar de forma secuencial y segura las peticiones concurrentes de generación de gráficos, evitando conflictos y bloqueos de Matplotlib.

Visualizaciones y Métricas Generadas

  • Función Sigmoide: Gráfica demostrativa sobre el mapeo probabilístico y el umbral de decisión del modelo.
  • Dispersión de Clases: Gráfico bidimensional dinámico con las características predictoras seleccionadas (con jitter en 1D para evitar superposiciones).
  • Curva de Aprendizaje: Gráfico lineal que compara la exactitud de entrenamiento frente a la de prueba a medida que aumentan los datos, permitiendo diagnosticar sobreajuste.
  • Matriz de Confusión: Comparativa visual directa entre predicciones y clases reales de validación.
  • Métricas de Precisión: Cálculo de Exactitud (Accuracy), Precisión (Precision), Sensibilidad (Recall) y Valor F1 (F1 Score) con ponderaciones balanceadas.
